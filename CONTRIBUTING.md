## 🚧 Contribution Policy

This is a personal portfolio repository, so it is not accepting external code contributions, pull requests, or merge requests.

You may use this project for yourself, fork it, and adapt it for your own personal site. Please do not open pull requests or ask to merge changes into this repository.

## 🍴 How You Can Use This Project

You may:

- Use this repository for your own learning.
- Fork it for personal use.
- Adapt the structure, ideas, or implementation for your own portfolio.
- Report serious bugs or security issues through the proper channels.

Please do not:

- Open pull requests against this repository.
- Ask to merge your changes into this repository.
- Submit redesigns, content updates, feature additions, or dependency changes for this repository.
- Reuse the owner's personal content, images, branding, contact details, or identity as your own.

## ✨ Personal Use Guidance

If you fork or reuse this project, update it for yourself before publishing:

- Replace names, profile details, links, images, and contact information.
- Review the license before reusing code.
- Remove any content that is specific to the original owner.
- Test your version before deploying it.

## 🐛 Reporting Issues

For general non-security issues, you may open an issue only if it describes a problem with the public site or repository that the owner should know about.

For security vulnerabilities, do not open a public issue. Follow the instructions in `SECURITY.md`.

## 🧑‍💻 Maintainer Notes

All changes to this repository are made by the repository owner or trusted maintainers. External pull requests will be closed without merge.
