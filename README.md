## ✨ Features

- 🎨 **Discord-Inspired UI** - Modern Discord user profile themed container with custom badge systems.
- 📱 **Fully Responsive** - Flawlessly adapted layouts for desktop, tablet, and mobile browsers.
- ⚡ **Fast & Lightweight** - Built using pure HTML5, CSS3, and Vanilla JavaScript without bulky framework overhead.
- 📬 **Interactive Pages** - Responsive sub-pages including dynamic Skills catalog, Projects browser, and visual Image Gallery.
- 💬 **Live Presence Integration** - Direct connection with Discord via Lanyard API displaying real-time rich presence.
- 🎭 **SEO & Metadata Optimized** - Full OpenGraph tags for rich social media sharing previews.

## 🎯 Setup & Installation

### Clone the Repository
```bash
git clone https://github.com/SahooShuvranshu/SahooShuvranshu.github.io.git
cd SahooShuvranshu.github.io
```

### Run Locally
You can view the site by opening `index.html` in any browser or using a python local server:
```bash
python -m http.server 8000
```
Visit `http://localhost:8000`.

## 🎨 Customization Guide

### Profile Details
To customize the profile details, modify the user metadata inside `index.html` and `contact.html`:
* **Avatar URL**: Search for `.discord-avatar` and replace with your profile image.
* **Username**: Edit the `.discord-username` container.

### Social Links
All button references can be updated in `index.html` and `contact.html`:
```html
<a href="https://github.com/YourUsername" target="_blank">
<a href="mailto:your.email@example.com">
```
